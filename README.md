# RuCalculus

"入语言"是一种用中文表示lambda演算的函数式编程语言，大概对标wenyan-lang。  
该项目致力于让lambda演算基本符合中文（文言文）的语法结构，例如入语言中大量使用后缀表达式（如square x写作甲之平方）。  
项目刚刚启动，功能和文档尚不完全，欢迎贡献任何力量。

## Quick Start

使用入语言解释器：
```
RuCalculus 源代码.入
RuCalculus 源代码.入 -d   # 逐步打印演算过程
```

使用stack启动入语言解释器：
```
stack run -- 源代码.入
```

在[samples](./samples/)目录下可以查看现有的范例程序。
